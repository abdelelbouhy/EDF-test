# EDF typescript tech test
## Main commands

``` bash
# run run.sh file that will install nmps and run unit tests and execute
# server.ts that will call the functions from the exampl.ts file 
# (calls localhost server to get some data and work on it in 
# bookSearchApi class) use next steps do it one by one
./run.sh

```

``` bash
# Install the dependencies
npm  install

# Run the server.ts file will log function result to the terminal
npm start

# Run the unit tests
npm test

# run example file (server should be running from previous step)
npm run example
```
